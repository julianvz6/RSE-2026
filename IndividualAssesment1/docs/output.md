# Output

The pipeline generates the following files in the results directory (`prac_output/` by default):

## wordcloud.png

Keyword cloud generated from the abstracts of all analyzed papers.

## figures_per_article.png

Bar chart showing the number of figures detected in each article.

## links_per_paper.txt

Text file listing all external links found in each paper, grouped by article.

Example:

```
2601.05005v1 (8 links):
  - https://example.com/paper1
  - https://example.com/paper2
  ...
```
