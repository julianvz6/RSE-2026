# Individual Assessment 1 - Grobid Pipeline

Automated pipeline that extracts structured data from academic PDF articles using [Grobid](https://github.com/kermitt2/grobid) and produces visualizations.

## Features

| Output | Description |
|---|---|
| **Keyword Cloud** | Word cloud generated from all paper abstracts |
| **Figures Chart** | Bar chart showing the number of figures per article |
| **Links List** | All external links found in each paper |

## Project Structure

```
RSE-2026-Individual-Assessment-1/
├── articles/              # Input PDF articles (10 open-access papers)
├── grobid_output/         # TEI-XML files generated by Grobid
├── prac_output/           # Generated visualizations and links list
├── tests/                 # Unit tests
├── docs/                  # Documentation
├── main.py                # Pipeline entry point (all functions here)
├── setup.cfg              # Configuration file (paths, Grobid settings)
├── config.json            # Grobid server configuration
├── requirements.txt       # Python dependencies
└── README.md
```

## Setup

### 1. Clone the repository

```bash
git clone https://github.com/julianvz6/RSE-2026.git
```

### 2. Create and activate a virtual environment

```bash
python -m venv venv01
```

**Windows (PowerShell):**
```bash
.\venv01\Scripts\Activate.ps1
```

**Windows (CMD):**
```bash
.\venv01\Scripts\activate.bat
```

**Linux / macOS:**
```bash
source venv01/bin/activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Grobid server

A running [Grobid](https://github.com/kermitt2/grobid) server is required to process PDFs. You can run one locally with Docker:

```bash
docker run --rm -p 8070:8070 lfoppiano/grobid:0.8.1
```

Then update `config.json` with your server address. Pre-processed XML files are already included in `grobid_output/`, so this step is optional if you just want to run the analysis.

## Usage

```bash
python main.py
```

All paths and settings are read from `setup.cfg`.

### Output

The pipeline generates the following files in `prac_output/`:

- `wordcloud.png` — keyword cloud from paper abstracts
- `figures_per_article.png` — bar chart of figure counts
- `links_per_paper.txt` — list of links found in each paper

## Configuration

Edit `setup.cfg` to change paths or Grobid settings:

```ini
[paths]
input = ./articles
grobid_output = ./grobid_output
results = ./prac_output

[grobid]
config = ./config.json
workers = 4
force = true
```
